**Postgres DB Server**

This script install installs the following:-


- ruby 1.8.7 ree

- POSTGIS 2.0.2

- GEOS 3.3.8

- PROJ 4.8.0

- GDAL 1.9.2

- Postgresql 9.2.3

- Oniguruma

- Sphinx


To begin the install run the install.sh script from the scripts directory.

It's been tested on an EC2 instance running x64 Ubutnu 12.04 LTS.
